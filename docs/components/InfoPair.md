# Info Pair

This is a customizable info pair component

## Examples

<Doc-InfoPair/>

## Code Example
```html
<ao-info-pair label="'Info Label'">
  Information goes here
</ao-info-pair>
```

## Props

| Name         | Type     | Default | Description                                                           |
|:-------------|:---------|:---------|:----------------------------------------------------------------------|
| label | String, required | - | Text displayed as the info pair label. |
